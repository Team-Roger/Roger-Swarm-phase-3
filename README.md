# 🌌 Overview

Phase 3 represents a major evolution of **ROGER SWARM**, introducing an advanced spatial hologram interaction system.

Building on the gesture-controlled particle foundation from Phase 2, this stage combines real-time hand tracking, holographic rendering, particle physics, and intelligent gesture interaction to create a more immersive human–computer interface.

The system now allows users to interact with digital holographic objects naturally through hand movement, creating the feeling of controlling a virtual object in physical space.

# ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/Team-Roger/Roger-Swarm-Phase3.git
```

Enter the project folder:

```bash
cd Roger-Swarm-Phase3
```

Install dependencies:

```bash
python -m pip install -r requirements.txt
```

Download required models:

```bash
python download_model.py
```

Run the project:

```bash
python main.py
```
